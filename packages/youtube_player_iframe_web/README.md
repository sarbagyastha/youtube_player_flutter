# youtube_player_iframe_web

The web platform implementation of [`youtube_player_iframe`](https://pub.dev/packages/youtube_player_iframe).

---

## You probably don't need to add this directly

`youtube_player_iframe` automatically uses this package on web via Flutter's [federated plugin](https://docs.flutter.dev/packages-and-plugins/developing-packages#federated-plugins) mechanism. You only need to depend on it explicitly if you're:

- Building your own platform plugin that implements `youtube_player_iframe`
- Working inside this monorepo

For normal app usage, just add `youtube_player_iframe` and this package is picked up automatically.
